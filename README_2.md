# 🛸 도다리 2 (Dodari 2) — Apple Silicon Mac 전용

## 1. 프로젝트 개요 (Overview)

도다리 2는 초거대 언어 모델(LLM)을 활용하여 EPUB, PDF, TXT 문서를 문맥에 맞게 번역하는 고성능 멀티-도큐먼트 번역 시스템입니다.  
이 버전은 **Apple Silicon Mac 전용**으로 최적화되었습니다.

* **주요 기능:** 맥락 기반 다국어 문서 번역 (EPUB, **PDF**, TXT)
* **지원 언어:** 한국어 · 영어 · 일본어 · 중국어 · 프랑스어 · 이탈리아어 · 네덜란드어 · 덴마크어 · 스웨덴어 · 노르웨이어 · 아랍어 · 페르시아어 (**12개 언어**)
* **핵심 엔진:** Gemma 4 E4B (기본) / 31B (고사양) + **Apple MLX** 가속 (M1 / M2 / M3 / M4 / M5)
* **배포 형태:** Gradio 기반 웹 대시보드

---

## 2. 핵심 기술 스택 및 인프라 (Tech Stack)

### 2.1. MLX 추론 엔진

* **플랫폼:** Apple Silicon (M1 이상) + **MLX-VLM**
  - Unified Memory를 활용한 대규모 컨텍스트 처리
  - 4-bit / 8-bit 양자화 모델 선택 가능
  - Continuous Batching으로 병렬 배치 번역

### 2.2. 문서 처리 기술

* **시맨틱 PDF 번역:** **IBM Docling**을 통한 의미론적 구조 분석 및 EPUB 패키징
* **표/코드 레이아웃 보존:** 넓은 표(5열+)나 코드 블록을 PDF 원본에서 직접 크롭(`pypdfium2`)하여 이미지로 대체
* **EPUB v2 파이프라인:** 리프 노드 탐색 및 태그 보존 전략으로 제목, 목차, 각주까지 완벽 번역

---

## 3. 주요 비즈니스 로직

1. **지능형 언어 감지:** 파일 업로드 시 `langdetect` 기반으로 원본 언어를 자동 감지하고 최적의 목표 언어를 자동 추천합니다.
2. **문체 일관성 시스템:** 사용자가 **서술체(~다)** 또는 **경어체(~합니다)**를 선택할 수 있으며, 나레이션과 대화문을 구분하여 일관된 어투를 유지합니다.
3. **이중언어 표기 순서:** `번역문(원문)` (기본) 또는 `원문(번역문)` (학습자 모드) 선택. 파일명도 선택에 맞게 자동 동기화됩니다.
4. **AI 용어집:** 소설·전문서의 고유명사를 AI가 자동 추출하고, 문서 전체에서 일관된 번역어를 강제 적용합니다.
5. **로컬 저장:** 번역 결과는 `outputs/` 폴더에 저장되며, Gradio UI 하단에서 바로 다운로드할 수 있습니다.

## 4. 설치 및 실행 가이드 (Execution Guide)

### 4.1. 사전 준비 (Prerequisites)

| 항목 | 최소 (E4B 기본) | 권장 (31B 고품질) |
|:---|:---|:---|
| 칩 | Apple Silicon M1 이상 **필수** | M3 Pro / M4 Max 이상 |
| 통합 메모리 | **8GB** (E4B 모델) | **32GB** 이상 (31B 모델) |
| 저장 공간 | 10GB 이상 여유 (E4B ~3GB) | SSD 35GB 이상 (31B ~20GB) |
| macOS | Ventura 13.0 이상 | 최신 버전 권장 |
| Python | 3.11 이상 | — |

> **모델 선택 가이드:**
> - **E4B (기본값)** — 빠르고 쾌적한 번역. 16GB 이하 맥에서도 원활하게 동작합니다.
> - **31B (고품질)** — 깊은 문맥 파악과 최고 품질. 32GB 이상 맥 권장합니다.


### 4.2. 실행 방법 (Launch)

```bash
bash start_mac.sh
```

최초 실행 시 `dodari_env` 가상환경 구성 및 Gemma 4 E4B 모델 자동 다운로드(~3GB)가 진행됩니다.  
구동 완료 후 `http://127.0.0.1:7860` 접속. 상단에 **"✔ Apple MLX 고속 번역 엔진 활성화됨"** 이 표시되면 준비 완료입니다.
